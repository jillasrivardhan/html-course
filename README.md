# 📘 HTML Course

A beginner-friendly collection of HTML projects and examples designed to help learners understand the fundamentals of web development. This repository covers essential HTML concepts through practical examples and mini-projects.

---

## 📂 Project Structure

```
html-course/
│── basics.html          # HTML basics and syntax
│── forms.html           # HTML forms and input elements
│── lists.html           # Ordered, unordered, and description lists
│── nav.html             # Navigation bar example
│── new.html             # Practice HTML page
│── portfolio.html       # Simple personal portfolio webpage
│── practice.html        # HTML practice examples
│── problems.html        # Practice exercises
│── tables.html          # HTML tables
│── user_login.html      # User login form
│── videos.html          # Embedding videos in HTML
│── style.css            # Main stylesheet
└── practice.css         # Additional CSS styles
```

## ▶️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/jillasrivardhan/html-course.git
```

### 2. Navigate to the project folder

```bash
cd html-course
```

### 3. Open any HTML file

Simply double-click any `.html` file or open it with your preferred browser.

Alternatively, if you're using **Visual Studio Code**, install the **Live Server** extension and click **Go Live** to preview the pages.
